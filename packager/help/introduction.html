Introduction to the KATANA FloorBoard Editor

Description

The KATANA FloorBoard Editor is MIDI editing software for the Roland KATANA Guitar Synthesizer and is supported on specific Windows, Linux and Mac platforms. It provides KATANA owners with a convenient user interface to edit sounds and settings within the device and to store and recall those settings on their computer for backup purposes.

Where to get the KATANA FloorBoard Editor

To download the latest version and to learn more about the KATANA FloorBoard Editor, visit:sourceforge.net/projects/grfloorboard/
