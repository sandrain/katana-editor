Patch Text Summary Example

To generate a list of settings stored within a single KATANA patch,
from the main menu of the KATANAFxFloorBoard Editor,
select the patch you want the list for,
then select Tools > Patch Text Summary.
 A box opens with a text listing that you can select and copy for pasting into your favorite text editor, email, or wherever you like.
The following text is such an example.
