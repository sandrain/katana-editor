NS - NOISE SUPPRESSOR Editing

The NOISE SUPPRESSOR (some call it a noise gate) or NS module can be enabled/disabled and the threshold and release knobs set from the main panel in the KATANA FloorBoard Editor using its mini panel's ON/OFF switch and settings knobs.



Typically, a noise gate is used to silence the audio signal when it goes below a given threshold. This is especially good for high output situations where you don't want the guitar to start feeding back during quiet parts of a song, like when another players solos.

To make detailed changes to the settings (basically to see the numeric values in this case), click on the mini panel to open the editor's NOISE SUPPRESSOR edit panel, which looks like the following image:



Each patch can use a different NOISE SUPPRESSOR setting and you can use Assigns to assign parameters to a GK button or knob and to the KATANA's CTL and EXP Pedals. You can even control the many KATANA parameters from your DAW software using the parameters in the MIDI Implementation section of the owner's manual.

NOTE: Each effect has its own unique set of parameters with a unique set of knobs and parameter fields to configure it with a different appearance. Therefore, if you change from one to another effect TYPE, the knobs and sub-panels will change to support the selected effect's parameters. The number of possible combinations make it almost impossible to explain every parameter's capabilities in the online help. See the owner's manual for more detailed information.

The KATANA effects parameters are equivalent to those of many Roland or other stomp boxes, amps and rack mount effects. If you need to know how to adjust parameters for a given effect TYPE, see the owners manual or try one of these websites to see how your effects work or how your favorite players' use their effects:
