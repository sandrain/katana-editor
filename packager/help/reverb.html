REVERB - Reverb Effect Editing

The Reverb Effect module can be enabled/disabled, a different reverb effect type selected, and the time and level knobs set from the main panel in the KATANAFxFloorBoard Editor using its stomp pedals ON/OFF switch, drop-down effects type menu and settings knobs. One of seven reverb effect types can be selected.


To make detailed changes to the settings, double-click on the stomp pedal to open the editor's REVERB Effects edit panel, which looks like the following image:


You can select any one of the seven reverb effects by picking it from the TYPE field. You can use only one REVERB TYPE per patch. However, each patch can use a different REVERB setting and you can use Assigns to assign parameters to the KATANA's CTL and EXP Pedals. You can also use the other effects modules in the KATANA, such as the Multi Effects (FX1/FX2) to add a second delay module with entirely different parameters . You can even control the many KATANA parameters from your DAW software using the parameters in the MIDI Implementation section of the owner's manual.

NOTE: Each effect has its own unique set of parameters with a unique set of knobs and parameter fields to configure it with a different appearance. Therefore, if you change from one to another effect TYPE, the knobs and sub-panels will change to support the selected effect's parameters. The number of possible combinations make it almost impossible to explain every parameter's capabilities in the online help. See the owner's manual for more detailed information.

The KATANA effects parameters are equivalent to those of many Roland or other stomp boxes, amps and rack mount effects. If you need to know how to adjust parameters for a given effect TYPE, see the owners manual or try one of these websites to see how your effects work or how your favorite players use their effects:
  • Wikipedia article on effects units << for basic information on effects units of all types
  • GuitarGeek.com's Guitar Rigs section << see how your guitar heroes rigs are wired.
  • GuitarGeek.com's Rig Archives << many more here, very cool! Click the little GO! button at the right of your selected player.
If you want to sound like a player on one of these lists, try to setup your KATANA effects similar to their rigs.
